# Pizzaria Web - Protótipo de Sistema

## Sobre o Projeto
Este projeto consiste no desenvolvimento de um **protótipo de site para uma pizzaria**, com o objetivo de simular um sistema real de pedidos online.

A proposta é construir uma aplicação web que permita ao usuário visualizar o cardápio, selecionar produtos e futuramente realizar pedidos de forma prática e intuitiva.

Este projeto faz parte do meu processo de evolução como desenvolvedor, sendo continuamente aprimorado com novas funcionalidades ao longo do tempo.

---

## Objetivos
- Praticar conceitos de desenvolvimento web  
- Simular um sistema real de pedidos  
- Trabalhar com organização de código e boas práticas  
- Evoluir progressivamente o projeto com novas features  

---

## Tecnologias Utilizadas
- HTML5  
- CSS3  

> Tecnologias adicionais serão incorporadas conforme a evolução do projeto.

---

## Funcionalidades Atuais
- Exibição do cardápio  
- Interface visual da pizzaria  
- Estrutura inicial do sistema
